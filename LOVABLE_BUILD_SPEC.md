# Authoricy Intelligence Platform - Frontend Build Specification

**For: Lovable.dev**
**Version: 3.0 (Greenfield Intelligence + Competitor Curation)**
**Date: January 2026**

---

## Executive Summary

Build a **world-class SEO intelligence platform** for Authoricy that rivals SEMRush, Ahrefs, and SurferSEO. This platform handles TWO distinct user journeys:

1. **Established Domains** - Rich historical data, competitive analysis, optimization focus
2. **Greenfield Domains** - No SEO history, competitor-first intelligence, market entry strategy

**Design Standard:** USD 100M B2B SaaS (Linear, Notion, Figma quality)
**Backend API:** `https://authoricy-engine-production.up.railway.app`

### 10.3 Competitor Curation Flow

```typescript
// Curation flow state machine
type CurationState =
  | { status: 'loading' }
  | { status: 'reviewing', removedCount: number, required: number }
  | { status: 'validating' }
  | { status: 'error', message: string }
  | { status: 'complete', finalSet: Competitor[] };

// Must track removal progress
const canProceed = (state: CurationState) =>
  state.status === 'reviewing' && state.removedCount >= state.required;
```
